## 四、硬件架构（芯片级）

### 4.1 MCU 矩阵（国民技术 N32 系列）

Rover X1 采用了国民技术的三级 MCU 架构：

#### 4.1.1 主控大脑 — N32H7xx 系列

| 参数 | 详情 |
|------|------|
| **内核** | ARM Cortex-M7 + Cortex-M4 多核 |
| **主频** | 600 MHz / 300 MHz |
| **角色** | 运动控制中枢 — 底层实时协调与多关节协同调度 |
| **关键特性** | CORDIC 加速器、100ps 级超高分辨率定时器 |
| **模拟接口** | 5Msps ADC 差分运放 |
| **通信接口** | 8 路 CAN FD、千兆以太网、EtherCAT |
| **职责** | 上层应用逻辑、数据处理、人机交互 |

#### 4.1.2 关节控制 — N32H47x 系列

| 参数 | 详情 |
|------|------|
| **内核** | ARM Cortex-M4F |
| **主频** | 240 MHz |
| **角色** | 执行单元 — 精准驱动关节电机 |
| **关键特性** | 集成 CORDIC 数学加速（三角函数与坐标变换） |
| **模拟前端** | 4.7Msps ADC + 40MHz 运放（高集成模拟前端） |
| **定时器** | 125ps 高精度定时器，输出高质量 PWM |
| **通信接口** | 全管脚映射 UART / CAN-FD |
| **职责** | 伺服驱动、轨迹规划、电机转矩与位置控制 |

#### 4.1.3 辅助控制 — N32G45x 系列

| 参数 | 详情 |
|------|------|
| **内核** | ARM Cortex-M4 |
| **主频** | 144 MHz |
| **角色** | 辅助控制 — 终端交互与传感器管理 |
| **功耗** | 低至 90μA/MHz |
| **存储** | 512KB 加密 Flash，硬件 ECC 校验 |
| **安全** | 硬件密码引擎（国密 SM2/3/4 + 国际标准） |
| **外设** | 7 路 UART、4 路 I2C、多路 ADC/DAC |
| **职责** | 传感器接入、通信扩展、用户指令传输 |

### 4.2 电机系统

| 项目 | 详情 |
|------|------|
| **电机类型** | 有刷减速直流电机（Geared DC Motors） |
| **运动机构** | 轮腿混合式（Hybrid Wheel-Leg） |
| **关节数量** | 具体自由度数（DOF）未公开 |

### 5.3 AI 与算法能力

| 能力 | 实现方式 |
|------|---------|
| **SLAM / 路径规划** | 基于双视觉的实时环境建图与路径规划 |
| **视觉跟随** | 机载摄像头识别目标主体，设定距离跟随 |
| **障碍检测与避障** | 前向视觉感知，刹车停止机制 |
| **动态平衡** | 专用控制算法管理轮腿混合系统的步态和姿态 |
| **负载补偿** | 智能负重控制，补偿负载引起的重心偏移 |
| **仿生动作** | 内置动作包（抬爪、跳舞、挥手、跳跃等） |

### 5.4 开发接口

| 接口 | 详情 |
|------|------|
| **SDK/API** | Explorer 版本支持，可自定义步态模式、行为序列、交互响应 |
| **编程环境** | 图形化编程 + 代码编程 |
| **App 控制** | Wi-Fi 连接手机 App 远程控制 |
