| ~~H4~~ | ~~IMU 型号和性能？~~ | ~~高~~ | **已确认：有 IMU**（SDK Imu topic），具体型号待确认 |
| H5 | 电池电压和化学类型？ | **中** | 4000mAh 配什么电压，决定实际能量 |
| H6 | 是否有足端力传感器？ | **中** | SDK 可读取 ground reaction forces，可能有力传感器或电流估计 |

## 五、标杆产品快速对比

| 维度 | 宇树 Go2 | MIT Mini Cheetah | DOBOT Rover X1 |
|------|---------|-----------------|----------------|
| **定位** | 消费级 + 教育 | 学术开源 | 家庭消费级 |
| **价格** | ~$1,600 | 非卖品（开源） | ¥7,499 (~$1,030) |
| **重量** | ~15kg | ~9kg | 15.5kg / 18.5kg（轮式） |
| **主处理器** | RK3588S（A76×4+A55×4 + NPU） | 自定义板载计算机 | N32H7xx MCU + **大概率有独立 SoC**（gRPC/DDS 需 Linux） |
| **关节数** | 12 DOF | 12 DOF | **12 DOF**（已确认） |
| **电机类型** | 无刷 + 齿轮 | QDD 准直驱无刷 | 有刷减速直流（推测） |
| **LiDAR** | 有（4D LiDAR） | 无 | 无 |
| **视觉** | 超广角摄像头 | 无 | RGB + **深度摄像头**（已确认） |
| **SDK 架构** | SDK2（CycloneDDS） | 开源 C++ 框架 | **双层架构：高层 gRPC + 低层 CycloneDDS** |
| **ROS 支持** | 有（ROS2） | 社区移植 | **间接支持**（CycloneDDS 兼容，可桥接） |
| **SDK** | 开放（SDK2） | 开源 | Explorer 版开放（dobot_quad_sdk） |
| **独特卖点** | LiDAR + 强 SDK 生态 | 开源 + 学术影响力 | 轮腿混合 + 深度摄像头 + 价格低 |
